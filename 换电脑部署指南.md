# 换电脑后重新部署 OpenClaw

## 1. 安装 OpenClaw
在新电脑上安装 OpenClaw 客户端

## 2. 登录账号
使用飞书账号登录（与你之前相同的账号）

## 3. 下载配置文件
```bash
# 克隆仓库到工作目录
git clone https://github.com/17606141273/openclaw.git D:\AI_Files
```

## 4. 启动 OpenClaw
启动后会自动加载工作空间的配置文件

## 5. 验证
检查是否正常加载了你的身份和设置

---

## 更新备份
如果需要更新备份：
```bash
cd D:\AI_Files
git add .
git commit -m "更新备份"
git push
```
