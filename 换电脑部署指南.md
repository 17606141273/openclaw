# 换电脑后重新部署 OpenClaw

## 1. 安装 Node.js
确保新电脑已安装 Node.js（建议 v18+）

## 2. 安装 OpenClaw
```bash
npm install -g openclaw
```

## 3. 配置工作目录
设置工作目录为 D:\AI_Files（如果你想用其他目录也可以）

## 4. 下载配置文件
```bash
# 克隆仓库到工作目录
git clone https://github.com/17606141273/openclaw.git D:\AI_Files
```

## 5. 启动 OpenClaw
```bash
openclaw
```
启动后会自动加载工作空间的配置文件

## 6. 配置飞书
使用飞书账号登录，具体配置方式参考官方文档

---

## 更新备份
如果需要更新备份：
```bash
cd D:\AI_Files
git add .
git commit -m "更新备份"
git push
```
