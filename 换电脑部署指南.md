# 换电脑后重新部署 OpenClaw

## 目录说明
- **配置文件目录**：OpenClaw 默认工作空间（用户目录下的 `.openclaw/workspace`）
- **文件产出目录**：`D:\AI_Files`（工作期间生成的文件放在这里）

## 1. 安装 Node.js
确保新电脑已安装 Node.js（建议 v18+）

## 2. 安装 OpenClaw
```bash
npm install -g openclaw
```

## 3. 配置工作目录（可选）
如果需要设置文件产出目录，可以在 AGENTS.md 中修改 `D:\AI_Files`

## 4. 下载配置文件
```bash
# 克隆仓库
git clone https://github.com/17606141273/openclaw.git

# 把文件复制到 OpenClaw 工作空间（默认路径）
# Windows: C:\Users\你的用户名\.openclaw\workspace
# Mac: ~/.openclaw/workspace
```

## 5. 启动 OpenClaw
```bash
openclaw
```

## 6. 配置飞书
使用飞书账号登录，具体配置方式参考官方文档

---

## 更新备份
```bash
cd C:\Users\你的用户名\.openclaw\workspace
git add .
git commit -m "更新备份"
git push
```
